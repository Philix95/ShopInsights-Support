# ShopInsights — Support & Feedback

Willkommen im Support-Repository für **ShopInsights**, das Analytics-Plugin für Shopware 6.

## So kannst du uns erreichen

### Bug melden
Du hast einen Fehler gefunden? [Erstelle einen Bug Report](../../issues/new?template=bug_report.yml) — je mehr Details, desto schneller können wir helfen.

### Feature vorschlagen
Dir fehlt eine Funktion? [Schlage ein Feature vor](../../issues/new?template=feature_request.yml) — wir prüfen jeden Vorschlag gegen unsere Roadmap.

### Frage stellen
Du hast eine Frage zur Nutzung? [Stelle deine Frage hier](../../issues/new?template=question.yml) — oder schau zuerst in unsere [Dokumentation](../../wiki).

### E-Mail
Du hast keinen GitHub-Account? Schreibe uns an **support@shopinsights.de** — deine Nachricht wird automatisch als Issue erfasst.

## Roadmap

Unsere aktuelle Planung findest du im [Project Board](../../projects). Dort siehst du, welche Features geplant sind, woran wir gerade arbeiten und was bereits umgesetzt wurde.

| Phase | Inhalt | Status |
|-------|--------|--------|

| Phase 2 | Produkt-Performance, ABC-Analyse | 🔄 In Planung |

## Hinweise

- **Antwortzeiten**: Wir bemühen uns, innerhalb von 2 Werktagen zu antworten.
- **Sprache**: Deutsch und Englisch sind willkommen.
- **Vertraulichkeit**: Bitte teile keine sensiblen Shop-Daten (API-Keys, Passwörter, Kundendaten) in Issues.
