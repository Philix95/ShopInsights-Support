# Shop Insights – Dokumentation

**Shop Insights** ist ein Analytics-Plugin für Shopware 6, das ein interaktives Umsatz-Dashboard und einen flexiblen Report Builder direkt in die Shopware Administration integriert. Damit kannst du deine Bestelldaten nach beliebigen Kennzahlen, Zeiträumen und Segmenten auswerten – ohne externe Tools oder Datenexporte.

## Schnellstart

1. **Plugin installieren** – Lade ShopInsights herunter und lege es im Ordner `custom/plugins/ShopInsights` ab.
2. **Aktivieren** – Führe `bin/console plugin:install --activate ShopInsights` aus.
3. **Cache leeren** – `bin/console cache:clear` ausführen.
4. **Loslegen** – Öffne die Administration und navigiere zu **Katalog → Shop Insights**.

Eine ausführliche Anleitung findest du unter [Installation und Setup](installation-und-setup.md).
